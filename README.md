# CDB_Editors
There are two CDB Editors. Its recommended to use "CDBEditor.html".





Features
1) Automatic saving system
2) Organized Categories thats read from the CDB
3) Raw Json editor (Can be toggle On/Off)
4) Basic search function. 


Instructions
1) You need to select the cdb.json file that you want to edit.
2) Select the location to save it in (Just select the location that you selected it from)
3) Then just edit the values/raw json.
